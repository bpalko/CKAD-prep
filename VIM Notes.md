# Useful VIM
'/' - search the file for a key (/sdf will pull me to a text with sdf)
i - insert
esc - exit edit mode
':' - command prompt
:q! - quit without write
:wq - quit and write
'A' - append to end of line
:undo - undo
'dd' - change word
'cw' - change word
'dw' - delete word
'o' - open (append) blank line below current line (no need to press return)
